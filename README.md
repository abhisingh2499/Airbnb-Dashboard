# Airbnb Performance Dashboard

## 📌 Project Description
This project presents an interactive Power BI dashboard built to analyze Airbnb data. It focuses on understanding listings, reviews, pricing, and market trends across different cities to support data-driven insights.

---

## 🛠️ Tech Stack
- **Power BI** – Designed and developed interactive dashboards for data visualization  
- **Power Query** – Performed data cleaning, transformation, and preprocessing  
- **DAX (Data Analysis Expressions)** – Created calculated measures and KPIs for business insights  
- **Data Modeling** – Built relationships between tables to enable efficient analysis and filtering    

---

## 📊 Data Source
The dataset was obtained from **Kaggle** and consists of Airbnb listings, reviews, pricing, and property details across multiple cities, used for analysis and visualization.

---


## ⭐ Features / Highlights

- KPI cards displaying **total listings, cities, hosts, property types, and reviews**  
- Market share analysis by **city with cumulative percentage insights**  
- Trend analysis of listings over time to identify **growth, peak, and decline phases**  
- Price comparison across **property types (hotel room, entire place, shared room, private room)**  
- Customer review analysis to understand **review frequency and user behavior patterns**  
- Seasonality analysis showing **monthly review trends across different cities**  
- Host trust and verification analysis highlighting **profile verification and reliability metrics**  
- Interactive dashboard with **filters and slicers** for dynamic exploration of data    

---

## 🎯 Business Problem
Airbnb operates across multiple cities with diverse property types and customer behaviors. However, it is challenging to identify which locations perform best, how pricing varies across property types, and how customer engagement changes over time.  
The goal is to analyze Airbnb data to uncover trends, optimize pricing strategies, and understand customer behavior.
---

## ❓ Key Questions
- Which cities have the highest number of listings and reviews?  
- How does pricing vary across different property types?  
- What are the trends in listings over time and how were they impacted by COVID-19?  
- How does customer review frequency behave across listings?  
- Are there any seasonal patterns in customer reviews across cities?  
- What percentage of hosts are verified and how does it impact trust?   

---

## 📈 Insights
- Cities like **Paris, New York, and Sydney** dominate in terms of listings and reviews  
- **Entire place listings** have higher average prices compared to shared or private rooms  
- Listings experienced significant growth until 2015–2016, followed by fluctuations and a decline during COVID-19  
- Most customers tend to leave **only a few reviews**, indicating low repeat engagement  
- Strong **seasonal patterns** observed, especially in European cities during peak travel months  
- A majority of hosts are **verified**, contributing to higher trust and platform reliability
